<h4> Project structure </h4>
<ul>
<li>config: contains the project config classes</li>
<li>Domain: contains the project entity classes</li>
<li>Exception: contains the controller advice</li>
<li>mapper: contains the mapper responsible to convert entity to dto</li>
<li>repository: contains repositories</li>
<li>service/dto: contains dto classes</li>
<li>service/interfaces: contains service interfaces</li>
<li>service/impl: contains service implementations</li>
<li>web/rest: contains controllers</li>
</ul>
